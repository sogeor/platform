# spfs

## Введение

Представляет собой файловую систему, созданную специально для платформы.

## Структура

### Атрибут

Представляет собой атрибут, на который может ссылаться любое количество элементов. Если ни один элемент не ссылается на
атрибут, то выделенную под него память можно освободить. Если `type` равен `0`, то атрибут считается недействительным.
Также начало каждого атрибута соответствует `spfs_attribute_t`, а размер остальной части равен `size`.
