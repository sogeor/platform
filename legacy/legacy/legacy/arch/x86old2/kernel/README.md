arch/x86/kernel
===============

## Управление сборкой

Вы можете управлять сборкой с помощью переменной `ARCH_X86_KERNEL_MACROS`, которую можно определить в аргументах
командной строки Makefile. Если вы не определите её, то она автоматически будет определена макросами, которые
объявляются по умолчанию.
